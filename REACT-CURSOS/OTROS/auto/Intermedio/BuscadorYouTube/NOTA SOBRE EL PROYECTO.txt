Objetivo:
Desarrollar una aplicación que permita a los usuarios buscar videos en YouTube. 
La aplicación mostrará una lista de videos relacionados con la búsqueda, 
incluyendo miniaturas y títulos, y el usuario podrá seleccionar un video para reproducirlo.

Tecnologías a Utilizar:
React (se puede iniciar con Create React App)
Axios para solicitudes HTTP o el Fetch API nativo de JavaScript.
YouTube Data API V3 para las búsquedas de videos.
